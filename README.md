# Signup Page - HTML & CSS

A clean, responsive, and beginner-friendly Signup Page developed using **HTML5** and **CSS3**. This project demonstrates the fundamentals of front-end web development, including form creation, Flexbox-based layout design, modern CSS styling, and responsive user interface design.

## 🚀 Features

* Responsive design for different screen sizes
* User-friendly signup form interface
* Modern UI using HTML and CSS
* Flexbox for perfect center alignment
* Rounded corners and smooth box-shadow effects
* Interactive button hover animations
* Clean and well-structured code

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
